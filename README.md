# MOVIE-RECOMMENDATION-SYSTEM-WITH-PYHON
## Description
### This is a  project to build  a movie recommendation system using python.
### User will post his or her favorite movie and the system will recommend similar and or related movies based on several factors including;
 - title
 - casting 
 - actors
 - time of production

## Steps to follow
### Among the steps to follow inlclude; we will need the data which is already generated https://drive.google.com/file/d/1cCkwiVv4mgfl20ntgY3n4yApcWqqZQe6/view 
- we will perform some data pre-processing
- feature extraction from the data
- Use cosine similarity Algorithm
